# Site Content

This folder controls the editable parts of the website.

- Edit JSON here for page text, cards, links, sections, and navigation.
- Do not edit `src/pages/*.astro` for routine content changes.
- Run `npm run content:check` before publishing if you want a quick validation.
- Run `npm run publish -- "Update site content"` to build, commit, and push.

Files:

- `site.json`: global title, author, nav, footer, social links.
- `home.json`: homepage.
- `about.json`: About page.
- `projects.json`: Projects page.
- `life.json`: Life page.
- `links.json`: Links page.
